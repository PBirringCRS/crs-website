# Assets Directory

This directory contains the website's static assets including CSS and PDF files.

## Required Files

### Capability-Statement.pdf
Please upload your Copper Ridge Security capability statement PDF to this directory with the filename: `Capability-Statement.pdf`

The website references this file in multiple locations:
- Navigation menu "Capability Statement" button
- Home page CTAs
- About page CTAs
- Services page CTAs
- Past Performance page CTAs
- Contact page
- Footer links

### Optional: Logo Images
If you want to replace the text-based logo with image logos:
- Upload logo files (PNG format recommended with transparent background)
- Suggested filenames: `logo-primary.png`, `logo-footer.png`
- Update the HTML in each page's header section

## Current Files
- styles.css - Main stylesheet for the entire website
